FAQ

Questions, meet answers

›Does it support dark mode?

Yes! Click the moon/sun in the header. We save your preference to localStorage and respect system settings.

›Can I use this commercially?

Absolutely. Replace content, tweak styles, and ship. A tiny attribution in your code comments is appreciated but not required.

›Is it accessible?

We use semantic HTML, keyboard-friendly controls, & visible focus. Motion is toned down when users prefer reduced motion.
